**目标**：解决周视图看不到跨周项目全貌的痛点

### Phase C：效率工具（3 个子功能）

#### C1. 快速创建（用户需求 + AI 建议）

**目标**：减少重复录入，一条文本自动填充多个字段

**交互流程**：
```
┌─────────────────────────────────────────────┐
│  ⚡ 快速创建    [项目X 棚A 张导 视频  ] [解析]│
├─────────────────────────────────────────────┤
│  解析结果：                                   │
│  项目名称: 项目X                              │
│  场    地: 棚A                                │
│  导    演: 张导                               │
│  类    型: 视频                               │
│                                              │
│  📌 最近使用：棚A · 张导 · 视频               │
│                                              │
│  [确认创建]  [修改后创建]  [取消]              │
└─────────────────────────────────────────────┘
```

解析规则：
- 匹配已知场地 → 自动填入 location
- 匹配已知人名 → 自动填入对应职能
- 匹配项目类型关键词 → 自动填入 type
- 剩余文字 → 项目名称

### Phase D：集成与导出（3 个子功能）

#### D1. iCal 日历订阅（用户核心需求）

**数据流向**：排片系统 → 推送到外部日历（单向输出）

```
┌──────────────┐    GET /api/calendar.ics    ┌──────────────┐
│  排片系统     │ ─────────────────────────→ │  手机日历     │
│  (数据源)     │    Content-Type: text/cal  │ Outlook      │
│              │                            │ Google 日历   │
└──────────────┘                            └──────────────┘
```

用户在手机日历中添加订阅链接 → 日历自动显示排期项目 → 自动刷新

每个日历事件：
```
标题：项目名称（视频）
时间：当天 09:00-11:00（或全天事件）
地点：棚A
描述：导演: 张导 / 摄影师: 王摄 / 状态: 已确认
```
